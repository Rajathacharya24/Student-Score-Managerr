
# Student Score Manager

## Description

A small Java console application to manage and analyze student scores. You can enter student names and scores, view them, find the highest/lowest score, calculate the average score, and search for a student by name. 

---

##  Features

* Enter multiple students (name + score)
* Display a list of all students with their scores
* Find and show the highest score
* Find and show the lowest score
* Calculate and display the average score
* Search for a student by name

---

##  How to Run

1. Make sure **Java JDK** is installed.
2. Save your file as `StudentINF.java`.
3. **Compile** the program:

   ```
   javac StudentINF.java
   ```
4. **Run** the program:

   ```
   java StudentINF
   ```
